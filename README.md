# 🙏 Church Website

A beautiful, easy-to-manage website for your church community. Perfect for sharing service times, events, and building community connections.

## Features

✅ **Home Page** - Welcome visitors with service times and upcoming events preview
✅ **About Page** - Share your church's mission, history, and leadership
✅ **Events Page** - List and manage all upcoming church events
✅ **Contact Page** - Easy contact form for visitors to reach out
✅ **Responsive Design** - Looks great on phones, tablets, and computers
✅ **Modern Styling** - Professional, clean, and welcoming design

## What's Included

- `index.html` - Home page with welcome and service times
- `about.html` - Information about your church
- `events.html` - Full event listing and registration
- `contact.html` - Contact information and message form
- `styles.css` - All styling for the website
- `README.md` - This file

## Quick Start

1. **View the website locally:**
   - Open any HTML file in your web browser
   - Or double-click `index.html`

2. **Customize with your information:**
   - Open each `.html` file with a text editor
   - Replace placeholders with your church's info:
     - Church name
     - Address and contact details
     - Service times
     - Events
     - Leadership names

## How to Deploy (Free!)

### Using GitHub Pages (Easiest)

1. Your repository is already set up for GitHub Pages!
2. Go to your repository Settings → Pages
3. Select "main" branch as the source
4. Your site will be live at: `https://honzapaciorek.github.io/church-website/`

### Get a Custom Domain

1. Buy a domain (around $10-15/year from services like Namecheap, GoDaddy, etc.)
2. Point your domain to GitHub Pages
3. Update Settings → Pages with your custom domain

## Customization Guide

### Change Church Name
Search for "Our Church" in all `.html` files and replace with your church name.

### Update Contact Information
Find these sections and update:
- Phone number
- Email address
- Physical address
- Office hours

### Add/Edit Events
In `events.html`, find the event cards and modify:
- Event date
- Event title
- Time and location
- Description
- Registration details

### Change Colors
Edit `styles.css` and look for these color codes:
- `#2c3e50` - Dark blue (headings)
- `#3498db` - Light blue (buttons)
- `#667eea` - Purple (hero section)
